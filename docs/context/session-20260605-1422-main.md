## Key Decisions

### ADR-0010 — Cut anchoring via stable block IDs (pivotal)
Array-index + char-offset anchoring shifts silently on every insertion/deletion. Stable monotonic `BlockID` on every `Block` (minted by `Document.mintBlockID()`, never reused) makes cuts position-independent of prose edits within a block and immune to block reordering. This constrains all future cut-related code to use `BlockID`, never array index.
